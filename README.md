# Proyecto 6: Computer Vision
Es un proyecto simple que busca ser un acompaniante de un estudiante o programador,usa vision artificial para guardar perfiles,si asi lo desea el usuario, y ver las emociones que tuvo el usuario durante su session
El programa gaurda toda la informacion de la sesion en un json que despues el usuario puede exportar si quiere hacer un analisis mas detallado 
Uso:
El programa corre en la terminal de python al iniciar,se abre un menu con el cual puedes iniciar tu sesion,esto abre la camara,ver como va la sesion,agregar,quitar o ver perfiles y guardar y salir
